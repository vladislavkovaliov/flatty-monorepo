# TanStack

> TanStack is a set of open-source TypeScript libraries for routing, async state, server state, tables, forms, virtualization, data stores, full-stack apps, AI SDKs, and developer tooling.

This file routes agents to the smallest useful TanStack docs source. Fetch the linked product docs index first, then follow links from that page when you need deeper context. Root `/llms-full.txt` intentionally returns this same index because the complete TanStack docs corpus is too large for a useful root-level context file.

## Framework docs

- [TanStack Start](https://tanstack.com/start/latest/llms.txt): Full-stack Framework powered by TanStack Router for React and Solid. Frameworks: React, Solid. Packages: @tanstack/start-client-core. GitHub: https://github.com/tanstack/router.
- [TanStack Router](https://tanstack.com/router/latest/llms.txt): Type-safe Routing for React and Solid applications. Frameworks: React, Solid. Packages: @tanstack/router-core, react-location. GitHub: https://github.com/tanstack/router.

## Data & State Management docs

- [TanStack Query](https://tanstack.com/query/latest/llms.txt): Powerful asynchronous state management, server-state utilities and data fetching. Frameworks: React, Preact, Solid, Vue, Svelte, Angular, Lit. Packages: @tanstack/query-core, react-query. GitHub: https://github.com/tanstack/query.
- [TanStack DB](https://tanstack.com/db/latest/llms.txt): The reactive client-first store for your API. Frameworks: React, Vue, Solid, Svelte, Vanilla. Packages: @tanstack/db, @tanstack/react-db. GitHub: https://github.com/tanstack/db.
- [TanStack Store](https://tanstack.com/store/latest/llms.txt): Framework agnostic data store with reactive framework adapters. Frameworks: React, Preact, Solid, Svelte, Vue, Angular, Lit. Packages: @tanstack/store. GitHub: https://github.com/tanstack/store.
- [TanStack AI](https://tanstack.com/ai/latest/llms.txt): A powerful, open-source AI SDK with a unified interface across multiple providers. Frameworks: React, Vue, Solid, Svelte, Preact, Vanilla. Packages: @tanstack/ai-client. GitHub: https://github.com/tanstack/ai.

## UI & UX docs

- [TanStack Table](https://tanstack.com/table/latest/llms.txt): Headless UI for building powerful tables & datagrids. Frameworks: Angular, React, Preact, Solid, Svelte, Vue, Qwik, Lit, Alpine, Vanilla. Packages: @tanstack/table-core, react-table. GitHub: https://github.com/tanstack/table.
- [TanStack Form](https://tanstack.com/form/latest/llms.txt): Headless UI for building performant and type-safe forms. Frameworks: React, Preact, Vue, Angular, Solid, Lit, Svelte. Packages: @tanstack/form-core. GitHub: https://github.com/tanstack/form.
- [TanStack Hotkeys](https://tanstack.com/hotkeys/latest/llms.txt): Type-safe keyboard shortcuts, sequences, and key state tracking for your apps. Frameworks: React, Preact, Solid, Svelte, Vue, Angular. Packages: @tanstack/hotkeys. GitHub: https://github.com/tanstack/hotkeys.

## Performance docs

- [TanStack Virtual](https://tanstack.com/virtual/latest/llms.txt): Headless UI for Virtualizing Large Element Lists. Frameworks: React, Solid, Vue, Svelte, Lit, Angular, Marko. Packages: @tanstack/virtual-core, react-virtual. GitHub: https://github.com/tanstack/virtual.
- [TanStack Pacer](https://tanstack.com/pacer/latest/llms.txt): Framework agnostic debouncing, throttling, rate limiting, queuing, and batching utilities. Frameworks: React, Preact, Solid, Angular, Vanilla. Packages: @tanstack/pacer, @tanstack/pacer-lite. GitHub: https://github.com/tanstack/pacer.
