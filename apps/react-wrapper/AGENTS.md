# react-wrapper

Vite shell host for micro-frontends.

- **Port:** 5174
- **Stack:** React, TypeScript
- **Bundler:** Vite
- **Role:** Host shell — loads remote MFs via `<script>` at runtime

## Commands

| Command | Description |
|---------|-------------|
| `npm run start` | Dev server (port 5174) |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
