# react-settings

Webpack UMD remote micro-frontend.

- **Port:** 8081
- **Stack:** React, TypeScript
- **Bundler:** Webpack
- **Role:** Remote MF — exposes `window.ext-apps.settings`

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Webpack dev server (port 8081) |
| `webpack --mode production` | Production build |
