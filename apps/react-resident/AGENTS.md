# react-resident

Webpack UMD remote micro-frontend.

- **Port:** 8082
- **Stack:** React, TypeScript, Mantine v8, TanStack Query
- **Bundler:** Webpack
- **Role:** Remote MF — exposes `window.ext-apps.resident`

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Webpack dev server (port 8082) |
| `npm run build:webpack` | Production build |
| `npm run lint` | ESLint |
| `npm test` | Vitest |
| `npm run api` | Generate API types from Swagger |
| `npm run graphql:codegen` | GraphQL codegen |
