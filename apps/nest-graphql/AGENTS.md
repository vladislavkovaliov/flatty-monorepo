# nest-graphql

GraphQL API built with NestJS.

- **Port:** 3000
- **Stack:** NestJS, GraphQL, auth guard
- **Entry:** `src/main.ts`

## Commands

| Command | Description |
|---------|-------------|
| `nx serve nest-graphql` | Start server with watch (`nest start --watch`) |
| `nx build nest-graphql` | Build (`nest build`) |
| `nx test nest-graphql` | Run tests (`npm test`) |
| `nx lint nest-graphql` | Lint (`npm run lint`) |
